# PLTGU WhatsApp Bot\n\nBot WhatsApp untuk mengumpulkan dan melaporkan aktivitas operasi PLTGU Blok 1 UP Muara Karang.\n\n## Setup di Termux\n\n### 1. Install Dependencies\n```bash\npkg update && pkg upgrade\npm install git nodejs\n```\n\n### 2. Clone Repository\n```bash\ngit clone https://github.com/scoout/pltgu-whatsapp-bot.git\ncd pltgu-whatsapp-bot\n```\n\n### 3. Install NPM Packages\n```bash\nnpm install\n```\n\n### 4. Setup Environment\n```bash\ncp .env.example .env\n```\n\n### 5. Run Bot\n```bash\nnpm start\n```\n\nBot akan menampilkan QR Code. Scan dengan WhatsApp untuk autentikasi.\n\n## Fitur Bot\n\n- ✅ Menerima pesan log operasi\n- ✅ Menyimpan log ke database JSON\n- ✅ Command `/laporan` untuk generate laporan harian\n- ✅ Format laporan sesuai template PLTGU\n- ✅ Laporan otomatis setiap hari (configurable)\n\n## Contoh Usage\n\nKirim pesan ke bot:\n````\n[15/3, 21:30] Indrawan: 21:42 Condensat Desal masuk Hotwell\n````\n\nBot akan otomatis menyimpan log ini.\n\nUntuk laporan harian, kirim:\n````\n/laporan\n````\n\nBot akan mengirim laporan operasi lengkap format PLTGU.\n\n## Commands\n\n- `/laporan` - Generate laporan harian\n- `/clear` - Hapus semua log\n- `/help` - Tampilkan bantuan
